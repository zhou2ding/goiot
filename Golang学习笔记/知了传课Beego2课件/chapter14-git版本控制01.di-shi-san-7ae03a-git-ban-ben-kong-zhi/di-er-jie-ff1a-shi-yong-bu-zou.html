使用git的步骤：

1.克隆项目到本地：    
    下载到本地：git clone -b branch_name http://112.126.70.68:8000/cus/store.git
2.用pycharm打开克隆的项目
3.拉取master分支代码
    vsc -->git -->pull -->取消自己的分支，选择mstaer分支初始化完成

项目中使用git：针对这个项目文件操作的

1.拉取代码前先保存本地的代码
    vsc --> git --> stash   命名为当前的时间：2207   22:07
2.拉取master分支代码
    vsc --> git --> pull    选择master分支

3.stash本地缓存的代码
    vsc --> git --> unstash   unstash前先勾选中pop，意思是unstash的同时，删除本地缓存的代码，不然会导致自己分不清什么时候unstash的代码

4.add
    vsc --> git --> add
5.commit  新的代码，new:新增的功能点    修改的代码:update:修改的功能点   修改bug的代码，bugfix:修改了那些bug
    vsc --> git --> commit     填写提交的日志，以及勾选要提交的文件，不想提价的不要勾选即可

6.push   push到自己的分支
    vsc --> git --> push        选择自己的分支，默认和拉取时候的分支一致，所以拉取的时候最好拉取自己的分支

7.merger   在gitlab中点击Merge Requests --> New merge request --> Source branch  下的select source branch 中选择自己的分支
    --> Compare branches and continue  -->   Submit merge request  --> merger
