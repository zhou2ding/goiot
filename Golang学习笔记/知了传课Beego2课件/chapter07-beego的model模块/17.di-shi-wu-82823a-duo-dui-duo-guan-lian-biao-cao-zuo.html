多对多

  • 多对多插入

    • orm接口

      • o := orm.NewOrm()

post := models.Posts{Name:"post",CreateTime:time.Now(),Content:"xxxx"}
tag := models.Tags{Name:"tag",CreateTime:time.Now(),DescTag:"tag"}
// 插入post和tag数据
o.Insert(&post)
o.Insert(&tag)

//多对多插入
m2m := o.QueryM2M(&post,"Tags")     //字符串是结构体名称
m2m.Add(&tag)
// 或者
m2m := o.QueryM2M(&tag,"Posts")

// m2m接口请看多对多操作接口章节
m2m.Add(&post)

// 可以使用已知的数据插入
post := models.Posts{Id:2}
tag := models.Tags{Id:2}
m2m := o.QueryM2M(&post,"Tags")
m2m.Add(&tag)

    • 原生sql：注意desc字段名不能使用，和字段描述的关键字冲突了，修改模型的字段

    • ret_post, _ := o.Raw("insert into post(name,create_time,content) values(?,?,?)","post测试",time.Now(),"post内容").Exec()
ret_tag, _ := o.Raw("insert into tag(name,create_time,desc_tag) values(?,?,?)","tag测试",time.Now(),"tag描述").Exec()
post_id,_ := ret_post.LastInsertId()
tag_id,_ := ret_tag.LastInsertId()
o.Raw("insert into post_tags(post_id,tag_id) values(?,?)",post_id,tag_id).Exec()

  • QueryM2Mer操作接口
    • Add 添加 看上面的插入部分
    • Remove 删除关联数据
      • o := orm.NewOrm()
tag := models.Tags{Id:6}
m2m := o.QueryM2M(&tag,"Posts")
// 删除tagid为6绑定的指定帖子
post := []models.Posts{{Id:5},{Id:1}} // 这里可以是切片，也可以是单条数据，但是必须指定至少一个

m2m.Remove(&post)
    • Clear 清除所有 M2M 关系，和remove的区别，不需要传递删除的post_id，删除全部关联数据
      • o := orm.NewOrm()
tag := models.Tags{Id:6}
m2m := o.QueryM2M(&tag,"Posts")
num,_ := m2m.Clear()
    • Count 统计关联数量
      • o := orm.NewOrm()
tag := models.Tags{Id:6}
m2m := o.QueryM2M(&tag,"Posts")
num,_ := m2m.Count()
fmt.Println(num)
    • Exist 判断是否存在
      • //查询多对多中的指定数据是不是存在
o := orm.NewOrm()
tag := models.Tags{Id:6}
post := models.Posts{Id:5}

m2m := o.QueryM2M(&tag,"Posts")
is_exits := m2m.Exist(&post)
fmt.Println(is_exits)

  • 多对多更新：

    • orm接口：_注意多对多需要多个字段的值同时定位一条数据_
      post_id := 3
old_tag_id := 4
new_tag_id := 2
o.QueryTable("post_tags").Filter("post_id",post_id).Filter("tag_id",old_tag_id).Update(orm.Params{
    "tag_id": new_tag_id,
})

    • 原生sql

      • post_id := 3
old_tag_id := 4
new_tag_id := 2
o.Raw("update post_tags set tag_id=? where post_id=? and tag_id=?",new_tag_id,post_id,old_tag_id).Exec()

  • 多对多删除

    • orm接口：看QueryM2Mer接口的remove和clear

    • 原生sql：和一对一的删除类似

  • 多对多查询

    • orm接口

      • 查询post上都有哪些tag，知道post_id

          tags := []models.Tags{}
  post_id := 1
  o.QueryTable("tag").Filter("Posts__post__id",post_id).All(&tags)
  fmt.Println(tags)

        查询tag都绑在了哪些post上，知道tag_id 关系名__表名__字段名

        • posts := []models.Posts{}
tag_id := 5
// Tags__tag__id ：关系名__表名__字段名
o.QueryTable("post").Filter("Tags__tag__id",tag_id).All(&posts)

        LoadRelated

        • tag_id := 5
tag := models.Tags{}
o.QueryTable("tag").Filter("id",tag_id).One(&tag)
num,_ := o.LoadRelated(&tag,"Posts")
fmt.Println(num)
for _,post := range tag.Posts {
    fmt.Println(post.Name)
}

      • 查询所有的

        • o:= orm.NewOrm()
posts := []models.Posts{}
o.QueryTable("post").All(&posts)
for k,_ := range posts{
    o.LoadRelated(&posts[k],"Tags")
}

fmt.Println(posts)
m.Data["posts"] = posts

    • 原生sql：查询中间表，使用QueryRows
