使用原生sql查询

  • 代码示例

    • 第一步：NewOrm

      • o := orm.NewOrm()

    • 第二步：Raw执行sql语句，返回RawSeter对象

      • var r RawSeter

      • r = o.Raw("UPDATE user SET name = ? WHERE name = ?", "testing", "slene")

    • 第三步：RawSeter调用执行的方法(Exec)并返回 sql.Result 对象：

      • ret,err : = r.Exec()

    • 注意：第二步和第三步可以合并为一步，这样就不用了预先定义好RawSeter对象了

      • ret,err : = o.Raw("UPDATE user SET name = ? WHERE name = ?", "testing", "slene").Exec()

    • 使用sql.Result 对象获取数据

      • fmt.Println(ret.LastInsertId()) // 返回执行结果的id

      • fmt.Println(ret.RowsAffected()) // 受影响的行数

RawSeter的方法：

  • Exec() (sql.Result, error) 只能操作改变数据库数据，不能查询

    • 执行 sql 语句，返回 sql.Result 对象

    • 增加

      • ret,err := o.Raw("insert into user values(id,?,?,?)","xxx1",19,"xxx2").Exec()

    • 更新

      • ret,err := o.Raw("update user set name=?,age=?,addr=? where id=?","update_test",28,"市区测试",27).Exec()

    • 删除

      • ret,err := o.Raw("delete from user where id=?",26).Exec()

  • QueryRow(…interface{}) error 查询一条数据，需要传接收返回值的容器
    • var id int  
var name string  
var age int  
var addr string  
err := o.Raw("select name,age from user where id=?",21).QueryRow(&id,&name,&age,&addr)  
fmt.Println(err)  
fmt.Println(id)  
fmt.Println(name)  
fmt.Println(age)  
fmt.Println(addr)  

// 还可以使用结构体 ,推荐使用结构体
var user_raw models.UserStruct  
err := o.Raw("select name,age from user where id=?",21).QueryRow(&user_raw)  
fmt.Println(err)  
fmt.Println(user_raw)
  • QueryRows(…interface{}) (int64, error) 查询多条数据，参数是结构体切片
    • var user_raw []models.UserStruct
num, err := o.Raw("select * from user where name=?","11").QueryRows(&user_raw)
fmt.Println(err)
fmt.Println(num)

for _,v := range user_raw{
    fmt.Println(v.Name)
}
  • SetArgs(…interface{}) RawSeter

    • 改变 Raw(sql, args…) 中的 args 参数，返回一个新的 RawSeter

    • 用于单条 sql 语句，重复利用，替换参数然后执行

    • r := o.Raw("select * from user where name=?","11")
num , err := r.SetArgs("22").QueryRows(&user_raw)
fmt.Println(err)
fmt.Println(num)

for _,v := range user_raw{
    fmt.Println(v.Name)
}

    • 封装sql查询语句，使用SetArgs传参即可

      • func QueryRowsByName(name string)  ([]UserStruct){
    o := orm.NewOrm()
    query_rows_by_name := o.Raw("select * from user where name=?","11")
    var user_raw []UserStruct

    query_rows_by_name.SetArgs(name).QueryRows(&user_raw)
    return user_raw
}

// 调用
user_raw = models.QueryRowsByName("22")

for _,v := range user_raw{
    fmt.Println(v.Name)
}
  • Values(*[]Params, …string) (int64, error)

    • 返回结果集的 key => value 值

    • var params []orm.Params
num, err := o.Raw("select * from user where name=?","11").Values(&params)

for _, m := range params {
      fmt.Println(m["Id"], m["Name"])
      fmt.Println(m["Name"].(string))
  }
  • ValuesList(*[]ParamsList, …string) (int64, error)

    • 返回结果集 slice

    • var params []orm.ParamsList
num, err := o.Raw("select * from user where name=?","11").ValuesList(&params)
fmt.Println(num)
fmt.Println(err)
  • ValuesFlat(*ParamsList, string) (int64, error)

    • 返回单一字段的平铺 slice 数据

    • var list orm.ParamsList
num, err := o.Raw("SELECT id FROM user WHERE id < ?", 10).ValuesFlat(&list)
if err == nil && num > 0 {
    fmt.Println(list) // []{"1","2","3",...}
}
  • RowsToStruct(interface{}, string, string) (int64, error) 把数据当成key和value

    • 将查询结果转为结构体

    • 通过结构体的字段名获取对应的值

      • 假设数据库中的数据结构为:

      | id | name | value | | :--- | :--- | :--- | | 1 | total | 100 | | 2 | page | 10 |

    • 代码：

    • type Options struct {  

    Total string  
    Page  int
}

user_raws := new(Options)
num, err := o.Raw("select name,value from to_struct").RowsToStruct(user_raws,"name","value")
fmt.Println(user_raws.Total)
fmt.Println(user_raws.Page)

  • RowsToMap(*Params, string, string) (int64, error)

    • 将查询结果匹配到map里

    • 假设数据结构为下

    | id | name | value | | :--- | :--- | :--- | | 1 | total | 100 | | 2 | page | 10 |

    • res := make(orm.Params)
num, err := o.Raw("select name,value from to_struct").RowsToMap(&res,"name","value")
fmt.Println(num)
fmt.Println(err)
fmt.Println(res)
fmt.Println(res["total"])
fmt.Println(res["page"])
  • Prepare() (RawPreparer, error)

    • 用于一次 prepare 多次 exec，以提高批量执行的速度。

    • Prepare：参数是在RawPreparer的Exec中传参

    • p, err := o.Raw("UPDATE user SET name = ? WHERE name = ?").Prepare()

      res, err := p.Exec("testing", "slene")

      res, err = p.Exec("testing", "astaxie")

      ...

      p.Close() // 别忘记关闭 statement
