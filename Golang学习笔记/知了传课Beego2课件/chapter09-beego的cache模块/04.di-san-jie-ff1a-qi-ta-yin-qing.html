知了课堂beego框架课程

beego中缓存的引擎：

  • memory
    • cache.NewCache("memory",`{"interval":60}`)}
      • 过期时间：{"interval":60}
  • file
    • cache.NewCache("file",`{"CachePath":"./cache","FileSuffix":".cache","DirectoryLevel":"1","EmbedExpiry":"120"}`)
      • CachePath：缓存的文件目录
      • FileSuffix：文件后缀
      • DirectoryLevel：目录层级
      • EmbedExpiry：过期时间，字符串类型

  • redis

    • 安装：

      • go get github.com/gomodule/redigo/redis

    • ac,err := cache.NewCache("redis",`{"conn":"47.114.134.146:6379","dbNum":"0"}`)

      • key:redis链接的名称

      • conn：连接信息

      • dbNum:哪个库

      • password：redis如果有密码

    • 注意：在put添加数据的时候，timeout参数一定是time类型(time.Second*10)

  • memcache

    • 本地服务安装
      • 使用管理员身份打开cmd，进入存放memcached.exe文件的路经(切换盘符 d: 进入文件路径 cd )
      • 安装：memcached.exe -d install。
      • 启动：memcached.exe -d start。
      • 检查：搜服务，然后找memcached的服务，是不是状态为：正在运行
    • 源码包安装：
      • go get github.com/bradfitz/gomemcache/memcache
    • cache.NewCache("memcache",`{"conn":"127.0.0.1:11211"} ')
      • conn:memcache 的连接地址
