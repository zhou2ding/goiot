表设计

字段名称 类型 长度 说明
id int 11 主键，自增长
uid int 11 用户id外键
flag int 11 1:车辆逾期，2:所有通知
title string 64 消息标题
content longtext 0 消息内容
read_tag int 11 1:已读，0：未读

当前日期大于车辆归还日期，就表示逾期，当逾期了就应该保存一条数据到通知表中
