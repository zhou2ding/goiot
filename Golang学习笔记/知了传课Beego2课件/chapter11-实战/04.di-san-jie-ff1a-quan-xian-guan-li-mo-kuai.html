权限表设计：auth/resource/menu

字段名称 类型 长度 说明
id int 11 主键，自增长
auth_name string 64 权限名称
url_for string 255 菜单的url，一级菜单没有url，二级菜单才有
pid int 11 父级菜单的id
desc string 255 描述
cteate_time datetime 创建时间
is_active int 2 1:启用，0：停用
is_delete int 2 1:删除，0：未删除
weight int 11 权重,主要是为了菜单排序
