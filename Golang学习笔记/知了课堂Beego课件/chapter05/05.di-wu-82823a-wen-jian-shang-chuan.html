知了课堂Beego Web框架课程

文件上传之后一般是放在系统的内存里面，可以设置缓存中内存大小：

  • main.go中设置：

    • beego.BConfig.MaxMemory = 1<<22 单位是B

  • 配置文件中设置：

    • maxmemory = 1<<22 单位是B

  • 默认的缓存内存是64M

位移运算：

  • a<<b 表示把a转为二进制后左移b位（在后面添加 b个0） a*2^b --> 1*2^22

换算：

  • 1KB=1024B

  • 1M = 1024KB

  • 1GB = 1024MB

  • 1TB = 1024GB

内置方法：

  • GetFile(key string) (multipart.File, *multipart.FileHeader, error) 获取上传的文件

    • 接收一个参数，返回三个，filename是在FileHeader中

  • SaveToFile(fromfile, tofile string) error 保存上传的文件

    • 接收两个参数，第一个参数是表单的name参数，第二个是保存的路径，返回一个参数
    • 注意：这里的保存路径一定先创建好，否则会报错The system cannot find the path specified

示例代码：

type Upload struct {
    beego.Controller
}

func (u *Upload)Get()  {
    u.TplName = "upload_file.html"

}
func (u *Upload)Post()  {
    f,h,err := u.GetFile("upload_file")
    if err != nil {
        fmt.Println(err)
        return
    }
    defer f.Close()

    // 生成时间戳，防止重名
    timeUnix:=time.Now().Unix()       // int64类型
    time_str := strconv.FormatInt(timeUnix,10)   //  将int64转为字符串      convert：转换

    err1 := u.SaveToFile("upload_file","static/upload/"+time_str+h.Filename)            // 这里必须有这个路径才会保存成功， 否则报错：The system cannot find the path specified

    if err1 != nil{
        fmt.Println(err1)
    }
    u.TplName = "index.tpl"

}

注意：
相同文件名的文件会被后面上传的覆盖，可以使用时间戳重命名文件

前端：

<form action="/uploadFile" method="post" enctype="multipart/form-data">
    上传文件：<input type="file" name="upload_file">
    <input type="submit" value="提交">
</form>

注意：
这里必须设置enctype="multipart/form-data"，不然浏览器不会传送文件
